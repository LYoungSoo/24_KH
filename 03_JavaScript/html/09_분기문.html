분기문

    - 반복문 수행 중
      현재 반복을 멈추거나
      다음 반복으로 넘어가는 구문
    
    - break : (감싸고 있는 / 가까운) 반복문을 '멈춤'

    - continue : (감싸고 있는 / 가까운) 반복문의 시작 부분으로 이동 ==> 중간에 다음 반복으로 넘어감 ==> 한번'건너뜀'
