비교 연산자

  [자료형 관계 없이 비교 가능한 연산자]
    같다      : == (동등, 자료형 관계 없이 값이 같음), === (동일, 자료형과 값이 모두 같음)
    같지 않다 : != (자료형 관계 없이 값이 같지 않음) , !== (자료형 또는 값이 같지 않음)

  [숫자(number)만 비교 가능한 연산자]
    A >  B    : A가 B보다 크다        (A가 B 초과)
    A <  B    : A가 B보다 작다        (A가 B 미만)
    A >= B    : A가 B보다 크거나 같다 (A가 B 이하)
    A <= B    : A가 B보다 작거나 같다 (A가 B 미만)
  
    - 좌변, 우변을 ==> 같다, 같지않다, 초과, 미만, 이상, 이하 를 비교해서 맞을경우 true, 아닐 경우 false를 반환한다!!
      (boolean == 논리형 == true/false)

%(나머지), 비교 연산자를 이용해서 짝수 / 홀수, 배수 확인하기

홀 / 짝 판별기

숫자 입력 :

배수 판별기

은(는) 의 배수가 맞는가?

의 약수 목록 :

복합 대입 연산자(+=, -=, *=, /=, %=) (Assignment Operators)

    - 기존 변수에 저장되어 있는 값에 새롭게 대입되는 값의 산술 연산 결과를 대입하는 연산자
    - 코드가 짧고, 속도가 우위를 가지고 있음
    ex) let temp = 10;
        temp += 5; // temp = temp + 5;

논리 연산자( &&, ||, ! )

    
&& (AND) 연산자 ==> A && B

    - A 와 B 모두 true 일 경우에만 A && B 결과가 true인 연산자 (나머진 false)
    - 글 / 말 / 문장 에서 ~와, 그리고(이고), 이면서(면서), ~까지, ~부터, ~사이 라는 글귀가 있으면 AND 연산을 작성

    ex) 104는 100 이상의 숫자 이면서 짝수인가?
    ->   (104 >= 100)  &&  (104 % 2 == 0)  -> true
  
  
|| (OR) 연산자 ==> A || B

    - A 와 B 모두 false 일 경우에만 A || B 결과가 false 연산자 (나머진 true)
    - 글 / 말 / 문장 에서 ~또는, 이거나(거나), 하나라도 라는 글귀가 있으면 OR 연산을 작성

    ex) 4는 10을 초과하거나 짝수인가?
        ==> ( 4 > 10 ) || ( 4 % 2 === 0 );
              false    ||       true      ==> true
  
  
!(NOT) 연산자 ==> !A

  - 논리 값을 반대로 바꾸는 연산자

  !true   ==>  false
  !false  ==>  true

  ex)
  !(true && false) ==> true

  (10 > 5) || !(10 % 2 === 0) ==> true
  (10 < 5) || !(10 % 2 === 0) ==> false

  !!!true ==> false
