Window 내장 객체

    - 창(브라우저) 자체를 나타내는 객체
    - 자바스크립트 모든 객체의 최상위 부모
    - wondow의 자식 객체
      1) DOM (Document Object Model) : document ==> 브라우저에 내용이 출력되는 부분
      2) BOM (Browser Object Model)  : location, history, screen, navigator
          ==> DOM을 제외한 브라우저 나머지 부분 
    
    - JavaScript에서 제공하는 기본 함수, 객체 들은
      모두 window(브라우저 창) 안에서 호출되고, 수행되기 때문에 window 단어를 생랼할 수 있다.

      ex) window.alert() ==> alert
          window.prompt() ==> prompt
          window.document ==> document

시간 지연 함수

    - window가 제공하는 시간 지연 함수
      1) [window.]setTimeout(함수, 지연시간) 
        ==> 지연시간이 지난 후 함수를 실행 (1회)
      2) [window.]setInerval(함수, 지연시간)
        ==> 지연시간이 지날 때 마다, 함수를 수행 (계속 실행)
    - 시간 지연 함수는 모두 *비동기* 로 동작

비동기(Asynchronous)란?

  - 코드 해석(작업)이 작성 순서대로 수행되지 않고
    여러 작업이 동시 수행되는 것
  

  ==> 먼저 읽은 코드가 수행이 완료되는 것을 기다리지 않고
      바로 다음 코드를 수행하는 것
  

  [참고] 동기
  - 앞에 코드가 수행이 종료 되어야 다음 코드 수행

setTimeout(함수, 지연시간 (ms))

    - 지연시간이 지난 후 함수를 1회 실행
    - 지연시간은 ms 단위로 작성(ms == 1/1000초)

setInterval(함수, 지연시간ms)

    - 지연 시간이 지날 때 마다 함수를 수행
    * setInterval()을 종료시키기 위한 함수로 clearInterval()이 존재
