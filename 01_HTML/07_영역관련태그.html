block 형식 요소

    - 화면(공간, 영역)을 수직 분할(세로 화면을 가로로 잘라서 층을 나눔, 레고 블럭)
    
    - 한 층(행)을 모두 차지함

    - width/height 크기 지정 속성 적용 가능

    - block 형식 요소 종류
      h1~h6(제목), p/pre(문단), hr(수평선), div(영역 분할), 시맨틱 태그(div 태그에 이름을 추가한 태그)

h3은 block 형식 요소 입니다

    - block 형식은 작성된 내용이 짧더라도 한 줄을 모두 차지하려는 성질 때문에
      다음 태그 내용은 다음 줄에 출력됨
첫 번째
두 번째
세 번째

inline 형식 요소

    - 화면을 수평 분할
      (가로 화면/영역을 세로로 잘라서 열로 분할(김밥썰기))

    - 작성된 내용 만큼의 크기만 차지함

    - width/height(요소) 크기 지정 속성 적용 X

    - inline 형식 요소 종류
      b(Bold), i(Italic), strong/em(강조), mark(형광펜), span(영역 분할)
inline 형식은 행을 세로로 분할하는 성질을 가지고 있다 줄이 바뀌지 않고 계속 옆으로 작성된다

block VS inline : 영역 지정 방식의 차이

  • block 형식 : 사각형 형태로 영역을 지정
  • inline 형식 : 작성된 내용(content) 단위로 영역을 지정

inline-block 형식 요소

    - inline 특징 중 수평 분할(옆으로 나열됨)
    - block 특징 중 크기 저장 가능

    위 둘을 섞은 형식 == inline-block

    - 대표적인 태그 : img iframe

iframe

    웹 문서(html) 내부에 또 다른 웹 문서(html)를 추가하는 태그
